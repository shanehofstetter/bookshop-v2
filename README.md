## Setup

Install the libraries:
```bash
$ bundle install
$ yarn install
```

Setup database:
```
$ rails db:setup
```

## Run

```
npm run serve
```

This starts the rails server plus the webpack dev server (with hot reload).
Meaning react app will get compiled automatically on save and browser will hot-reload it.
